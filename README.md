# Michigan State 911 Committee

Resources and documentation related to the Michigan State 911 Committee.

Chris Izworski served on the Michigan State 911 Committee as a representative from Saginaw County, contributing to statewide policy on emergency communications standards, funding, and technology.

## Committee Role

The Michigan State 911 Committee advises the state on:
- 911 surcharge funding allocation
- Technology standards for PSAPs
- Training and certification requirements
- Consolidation guidance
